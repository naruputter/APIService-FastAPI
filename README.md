# API Service With FastAPI
## Requirement

- Python Version > 3.7

## Library

Install FastAPI library for create API service 

```bash
# fastapi
pip install fastapi
```

Install uvicorn library for run fastAPI service 

```bash
# uvicorn
pip install uvicorn
```

## Start

run python file name "run_service.py"
```bash
python run_service.py
```
